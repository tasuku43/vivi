# Non-goals

The following are intentionally out of scope unless future product docs explicitly change:

- File editing.
- Hosted service mode.
- Public network serving by default.
- Authentication and user accounts.
- Collaboration.
- Persistent indexing.
- Full IDE features.
- Git staging or commit workflows.
- Cloud storage integrations.
- LLM summarization or chat.
- Static-site generation.
- Replacing a documentation framework.
- Remote execution.

These exclusions keep the product small, local, safe, and focused on live viewing.
