Classic explorer + command palette

This mock keeps the standard file tree and tabbed main area, but adds a familiar command palette overlay for fast open, jumping to files, and action dispatch.

How it should feel

  • Sidebar remains the primary spatial navigator.
  • Tabs represent the working set of opened files.
  • Command+K opens a global palette without changing the underlying layout.
  • The palette is good for file open, recent files, renderer switching, and view actions. 
# example interactions
⌘K → "open read"     → open README.md tab
⌘K → "html preview"  → switch viewer mode for index.html
⌘K → "toggle raw"    → change markdown/code viewer state

This is the direction you described: classic as the base, command palette as a lightweight universal launcher.
