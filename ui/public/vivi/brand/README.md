# Vivi brand assets

The Open Reader mark represents the browser reading surface and the feedback
handoff between humans and coding agents. The cyan and amber tabs reuse Vivi's
selection and change colors.

- `vivi-icon.svg`: square service icon and favicon source.
- `favicon.ico`: 32px fallback favicon for older browser surfaces.
- `vivi-mark.svg`: transparent standalone mark for light surfaces.
- `vivi-lockup.svg`: transparent horizontal mark and wordmark for light surfaces.
- `vivi-readme.svg`: self-contained README header for light or dark GitHub themes.
- `vivi-icon-*.png`: raster service-icon sizes generated from `vivi-icon.svg`.
- `vivi-mark-512.png`: transparent raster mark generated from `vivi-mark.svg`.

The SVG files are the source of truth. Primary colors are `#07111a`, `#54c4dc`,
`#f0b35a`, and `#f7fafc`.
